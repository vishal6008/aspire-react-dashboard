# Aspire Card Dashboard

A React + TypeScript implementation of an Aspire-inspired card management dashboard.

## Live Demo

Hosted version of the project:

https://aspire-dashboard-rho.vercel.app/

---

## GitHub Repository

https://github.com/vishal6008/aspire-react-dashboard

---

## Tech Stack

* React 19
* TypeScript
* Vite
* Zustand
* CSS Modules

---

## Features

* Card carousel
* Add new card
* Freeze / unfreeze card
* Set spend limit
* Deactivate card
* Show / hide card number
* Responsive layout

---

## Installation

Clone the repository:

```
git clone https://github.com/vishal6008/aspire-react-dashboard.git
```

Go to the project folder:

```
cd aspire-react-dashboard
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

---

## Build for Production

```
npm run build
```

Preview build:

```
npm run preview
```
